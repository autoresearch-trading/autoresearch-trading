NEXT STEP: Run these commands to deploy
==========================================

1. Install Fly CLI:
   brew install flyctl
   flyctl auth login

2. Deploy:
   flyctl apps create pacifica-collector
   flyctl volumes create data_volume --region sjc --size 3 --app pacifica-collector
   flyctl deploy -c deploy/fly.toml --app pacifica-collector

3. Monitor:
   flyctl logs --app pacifica-collector

4. Sync data weekly:
   ./scripts/sync_cloud_data.sh
