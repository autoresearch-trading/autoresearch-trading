# Theorem 37: Rolling Z-Score Normalization Properties

## Definitions

Given a time series x_1, x_2, ..., x_N and a rolling window of size W:

**Rolling z-score normalization:**

  z_t = (x_t - mu_t) / sigma_t

where:
  mu_t = (1/W) * sum_{i=t-W+1}^{t} x_i  (rolling mean)
  sigma_t = sqrt((1/W) * sum_{i=t-W+1}^{t} (x_i - mu_t)^2)  (rolling std)

For t < W (warmup), use min_periods = P (minimum observations before computing).

**Rolling robust normalization (IQR-based):**

  z_t^robust = (x_t - median_t) / IQR_t

where:
  median_t = median of {x_{t-W+1}, ..., x_t}
  IQR_t = Q3_t - Q1_t (interquartile range)

**Clipping:** After normalization, z_t is clipped to [-C, C] (in our code, C = 5).

## Claims

### Part A: Z-Score Properties

1. **Centering.** The rolling mean of z_t over any window of size W is exactly 0 (when sigma_t > 0). That is, z-score normalization produces a zero-mean series within each window. Prove this.

2. **Unit variance (approximately).** The rolling variance of z_t over the same window is exactly 1 (when sigma_t > 0). Prove this.

3. **Order preservation.** Z-score normalization preserves ordering within a window: if x_a > x_b for a, b in the same window, then z_a > z_b. Prove this (follows from the transformation being strictly monotone for fixed mu, sigma).

4. **Stationarity.** If x_t is a stationary process with constant mean mu and std sigma, then z_t converges to (x_t - mu) / sigma as W → infinity. The rolling normalization approaches global normalization for stationary data. Prove this by the law of large numbers applied to the rolling statistics.

### Part B: Robust Normalization Properties

5. **Outlier resistance.** The breakdown point of the median is 50% (up to half the data can be corrupted without affecting the median). The breakdown point of IQR is 25%. For z-score, the breakdown point of the mean is 0% (a single outlier can shift it arbitrarily). Prove the median's 50% breakdown point: for any sample of size n, changing fewer than n/2 values cannot change the median by more than the gap to the next order statistic.

6. **IQR = 0 edge case.** When IQR = 0 (all values in the window are identical or nearly identical), our code replaces IQR with 1. Prove that this makes z^robust = x_t - median_t, which is a centering-only operation without scaling. T12 proved that IQR=0 makes the normalization non-injective (information loss).

### Part C: Clipping

7. **Clipping bounds.** After clipping to [-C, C], the normalized feature satisfies |z_t| <= C. Prove that clipping is idempotent: clip(clip(z, C), C) = clip(z, C).

8. **Clipping preserves ordering within bounds.** For z_a, z_b in (-C, C), clip(z_a) > clip(z_b) iff z_a > z_b. But for z_a > C and z_b > C, clip(z_a) = clip(z_b) = C (information loss). Prove both.

### Part D: Sufficiency Preservation (connecting to T12)

9. **Z-score preserves sufficient statistics (T12 reproved).** If T(x) is a sufficient statistic for parameter theta under density f(x; theta), and g(t) = (t - mu) / sigma is the z-score transformation with sigma > 0, then g(T(x)) is also sufficient for theta. Prove this using the Fisher-Neyman factorization theorem: f(x; theta) = h(x) * k(T(x); theta), and since g is bijective, k(T(x); theta) = k(g^{-1}(g(T(x))); theta) = k'(g(T(x)); theta).

10. **Clipping can destroy sufficiency.** Clipping is not bijective (it maps an interval to a point). If the sufficient statistic T(x) takes values outside [-C, C], clipping destroys sufficiency. Prove this with a counterexample: two different x values map to the same clipped z, but have different likelihoods under the model.
