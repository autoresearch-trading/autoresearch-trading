# Theorem 19: ApproximationGainVsEstimationCost

Let H_simple and H_complex be two hypothesis classes with

  H_simple subset H_complex.

Assume the expected loss at sample size n > 0 follows the stylized bias-variance model

  L_simple(n)  = B_simple + V_simple / n,
  L_complex(n) = B_complex + V_complex / n,

where

  B_simple  = bias^2(H_simple),
  B_complex = bias^2(H_complex),
  V_simple  > 0,
  V_complex > 0.

Assume also that

  B_complex < B_simple
  and
  V_complex > V_simple.

Prove the following claims.

1. Under the model above, the expected loss decomposes into a bias term plus an estimation term:

     E[loss(H)] = bias^2(H) + variance(H, n),

   with variance(H, n) of the form V(H) / n.

2. H_complex has lower approximation error but higher finite-sample variance:

     B_complex < B_simple
     and
     V_complex > V_simple.

3. H_complex beats H_simple exactly when n exceeds the critical sample size

     n_critical = (V_complex - V_simple) / (B_simple - B_complex).

   More precisely,

     L_complex(n) < L_simple(n)
     if and only if
     n > n_critical.

4. MLP versus MLP+TCN specialization. Suppose

     V_simple  = lambda * d,
     V_complex = lambda * (d + Delta),

   where d > 0 is the MLP parameter count, Delta > 0 is the additional parameter count contributed by the TCN, and lambda > 0 is a variance-per-parameter constant. Let

     g = B_simple - B_complex > 0

   be the approximation gap. Prove that

     n_critical = lambda * Delta / g
                = lambda * d * (Delta / d) / g.

   Conclude that the critical sample size grows linearly in the complexity increment Delta and also linearly in the complexity ratio Delta / d when d and g are fixed.
