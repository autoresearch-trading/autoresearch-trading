# Theorem 43: Conditional Spread and Slippage Under Activity Regimes

## Definitions

Let S(t) > 0 be the bid-ask spread at time t.
Let sigma(t) > 0 be the realized volatility at time t.
Let sigma_0 > 0 be the baseline (median) volatility.

Define the activity regime indicator:
  High activity: sigma(t) >= sigma_75  (75th percentile of volatility)
  Normal activity: sigma(t) < sigma_75

The conditional spread ratio is:
  R = E[S | high activity] / E[S]

The Almgren-Chriss power law model for spread-volatility relationship:
  E[S | sigma] = c * sigma^gamma

for some constant c > 0 and exponent gamma > 0.

## Claims

1. **Power law spread ratio.** Under the Almgren-Chriss model with exponent gamma:
     R = E[sigma^gamma | high activity] / E[sigma^gamma]

   For the special case where sigma follows a log-normal distribution, prove that R > 1 when gamma > 0 (spreads widen during high-activity periods).

2. **Square-root law (gamma = 0.5).** Under the classical square-root impact model:
     E[S | sigma] / E[S | sigma_0] = sqrt(sigma / sigma_0)

   If high-activity periods have sigma = k * sigma_0 for some k > 1, then:
     R_sqrt = sqrt(k)

   For k = 2 (doubled volatility): R_sqrt = sqrt(2) ≈ 1.414.
   For k = 4: R_sqrt = 2.
   Verify these numerical values.

3. **Linear bound.** For any gamma in (0, 1], the conditional spread ratio satisfies:
     1 < R <= k^gamma

   when high-activity volatility is at most k times normal volatility. Prove this upper bound.

4. **When R = 1 (no conditional widening).** R = 1 if and only if S is independent of the activity regime. This occurs when:
   a) The market maker maintains constant spreads regardless of volatility (institutional commitment), OR
   b) The activity measure does not correlate with the information asymmetry that drives spread widening, OR
   c) gamma = 0 (spread is constant)

   Our empirical finding: R = 1.00 across 23 symbols at the 100-trade batch timescale (~1-2 seconds). This is consistent with interpretation (a) or (b) — at sub-second timescales on a DEX, spreads are set by limit orders that persist across activity regimes.

5. **Slippage model correctness.** Given R = 1, the static slippage model:
     slippage = S/2 + impact_buffer

   is unbiased for E[slippage | signal]. If R > 1, the model underestimates by:
     bias = (R - 1) * S/2

   Prove that the relative bias is (R-1)/R of the true conditional slippage (excluding impact_buffer).
