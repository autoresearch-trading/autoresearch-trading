# Theorem 36: VWAP Batch Computation Correctness

## Definitions

Given a batch of B trades, each with price p_i > 0 and quantity q_i > 0 (i = 1, ..., B):

  VWAP = sum_{i=1}^B (p_i * q_i) / sum_{i=1}^B q_i

Define notional_i = p_i * q_i, total_notional = sum notional_i, total_qty = sum q_i.

The log return between consecutive batches:

  r_k = log(VWAP_k) - log(VWAP_{k-1})

The Trade Flow Imbalance (TFI) for a batch:

  TFI = (V_buy - V_sell) / (V_buy + V_sell)

where V_buy = sum_{i: side_i = buy} notional_i and V_sell = sum_{i: side_i = sell} notional_i.

## Claims

### Part A: VWAP Properties

1. **VWAP is a weighted mean of prices.** Prove that VWAP = sum_{i=1}^B w_i * p_i where w_i = q_i / total_qty and sum w_i = 1, w_i > 0. Therefore:

     min(p_1, ..., p_B) <= VWAP <= max(p_1, ..., p_B)

   VWAP is bounded by the price range of the batch.

2. **VWAP is monotone in prices.** If all prices increase by delta > 0 (p_i' = p_i + delta), then VWAP' = VWAP + delta. Prove this.

3. **VWAP is invariant to quantity scaling.** If all quantities are scaled by c > 0 (q_i' = c * q_i), then VWAP is unchanged. Prove this.

4. **VWAP with equal quantities reduces to arithmetic mean.** If q_i = q for all i, then VWAP = (1/B) * sum p_i. Prove this.

### Part B: TFI Properties

5. **TFI is bounded.** Prove that -1 <= TFI <= 1 always, with:
   - TFI = 1 iff V_sell = 0 (all buys)
   - TFI = -1 iff V_buy = 0 (all sells)
   - TFI = 0 iff V_buy = V_sell (balanced)

6. **TFI is a function of the buy fraction.** Let f = V_buy / (V_buy + V_sell). Then TFI = 2f - 1. Prove this.

7. **Signed notional decomposition.** V_buy - V_sell = TFI * (V_buy + V_sell). The signed notional is the product of TFI and total volume. Prove this.

### Part C: Return Computation

8. **Log return additivity.** For three consecutive batches with VWAPs v_1, v_2, v_3:

     r_{1→3} = r_{1→2} + r_{2→3} = log(v_3/v_1)

   Prove that multi-period log returns are additive (sum of single-period returns).

9. **Return from zero-movement batch.** If VWAP_k = VWAP_{k-1} (identical prices in both batches), then r_k = 0. Prove this.

### Part D: Batching Consistency

10. **Partitioning invariance.** Consider 2B trades split into two consecutive batches of B each, with VWAPs v_1 and v_2. The VWAP of the combined 2B trades satisfies:

      VWAP_combined = (Q_1 * v_1 + Q_2 * v_2) / (Q_1 + Q_2)

    where Q_1, Q_2 are total quantities in each half. Prove this, and prove that VWAP_combined is NOT generally equal to (v_1 + v_2) / 2 (simple average) unless Q_1 = Q_2.

    This matters because our fixed-size batching (B=100) means each batch has equal trade COUNT but NOT equal quantity. Adjacent batches are not interchangeable.
