# Theorem 18: FrequencyOptimumUnder5bps

Let c > 0 be the per-side transaction fee, so each trade pays round-trip fee 2c. For trade i, let

  mu_i = gross expected return of trade i,
  v_i  = downside variance contribution of trade i,
  a_i  = mu_i - 2c.

Assume v_i > 0 for all i. For n >= 1 define

  A_n = sum_{i=1}^n a_i,
  V_n = sum_{i=1}^n v_i,
  S_n = A_n / sqrt(V_n).

Interpret S_n as the unannualized Sortino score of the first n trades over a fixed horizon.

Prove the following claims.

1. For every n >= 1, adding trade n + 1 strictly improves Sortino, that is, S_{n+1} > S_n, if and only if

     mu_{n+1} > 2c + correction_term(n),

   where correction_term(n) depends on A_n, V_n, and v_{n+1}.

2. Derive the correction term explicitly as

     correction_term(n)
       = A_n * (sqrt(1 + v_{n+1} / V_n) - 1).

   Equivalently,

     S_{n+1} > S_n
     if and only if
     a_{n+1} > A_n * (sqrt(1 + v_{n+1} / V_n) - 1).

3. Homogeneous special case. Assume the first n trades all have the same net edge a > 0 and the same downside variance v > 0. Then

     A_n = n * a,
     V_n = n * v,
     correction_term(n) = n * a * (sqrt(1 + 1 / n) - 1).

   Prove that this break-even edge is strictly increasing in n and converges to a / 2 as n -> infinity.

4. Numerical claim under 5 bps per side. Take

     c = 0.0005,
     mu = 0.0015,
     a  = mu - 2c = 0.0005.

   In the homogeneous model of Claim 3, the break-even gross edge for trade n + 1 is

     mu_break(n) = 0.001 + 0.0005 * n * (sqrt(1 + 1 / n) - 1).

   Prove that mu_break(n) < 0.00125 for every n >= 1. Hence mu = 0.0015 is above the break-even threshold for every n, so under this homogeneous model there is no finite maximum profitable trade count in a 20-day period: every additional trade still increases S_n.
