# Gate 0 Random-Projection Control Baseline

**Method:** 83-dim flat features (mean/std/skew/kurt/last per channel, minus time_delta_last + prev_seq_time_span_last — session-of-day leak pruned 2026-04-23) → StandardScaler → FixedRandomProjection(seed=42, 83→20) → LogisticRegression(C=1.0).
Walk-forward 3-fold, 600-event embargo. Stride=200 for evaluation windows.

**Key difference from Gate 0 PCA:** The projection matrix is drawn once from a fixed seed and frozen for ALL folds and ALL symbols.  It is non-adaptive — it does not fit to training data.  Only StandardScaler and LR refit per fold.

**Purpose:** Noise floor.  If PCA substantially beats this, the 83-dim features carry real structure.  If they are similar, Gate 0 may reflect base-rate drift rather than learned structure.

**Metric:** Two tables reported per council-1 / council-5 review. Balanced accuracy is the council-preferred reading at ALL horizons. Raw accuracy preserved for historical comparison.
