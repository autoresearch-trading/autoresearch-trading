# Theorem 22: DualGateDependenceBound

Let G1 and G2 be Bernoulli gate indicators taking values in {0, 1}. Write

  p1 = P(G1 = 1),
  p2 = P(G2 = 1),
  p12 = P(G1 = 1 and G2 = 1).

Assume 0 < p1 < 1 and 0 < p2 < 1. Let rho = Corr(G1, G2).

Prove the following claims.

1. Exact Bernoulli correlation identity:

     p12 = p1 * p2 + rho * sqrt(p1 * (1 - p1) * p2 * (1 - p2)).

2. The combined pass rate exceeds the independence prediction exactly by the covariance term:

     p12 - p1 * p2 = Cov(G1, G2)
                   = rho * sqrt(p1 * (1 - p1) * p2 * (1 - p2)).

   In particular, if rho > 0 then p12 > p1 * p2.

3. Redundancy from a shared latent factor. Let T be a latent toxicity factor. Assume both gates are deterministic threshold functions of T:

     G1 = 1_{T <= t1},
     G2 = 1_{T <= t2},

   with t2 <= t1, so that passing G2 already implies passing G1. Prove that

     G1 and G2 are nested,
     p12 = p2,
     and conditioning on both gates is equivalent to conditioning on G2 alone.

   Conclude that in this nested same-factor model the second gate provides no additional information once the stricter gate is known.

4. Sortino redundancy threshold. Let S0 > 0 be the ungated Sortino. Suppose gate G1 alone gives

     S1 = k1 * sqrt(p1) * S0

   with k1 > 0. Suppose the dual-gated strategy has pass rate p12 and effective incremental accuracy factor k2_eff, so that

     S12 = k1 * k2_eff * sqrt(p12) * S0.

   Prove that the second gate improves Sortino over the first gate if and only if

     k2_eff * sqrt(p12 / p1) > 1.

   Assume in addition that k2_eff(rho) is nonincreasing in rho and that there exists rho_critical in [0, 1] satisfying

     k2_eff(rho_critical) * sqrt(p12(rho_critical) / p1) = 1,

   where p12(rho) is given by Claim 1. Then for every rho >= rho_critical, the second gate gives no additional Sortino improvement.

   In the nested same-factor case of Claim 3, one has k2_eff = 1 and p12 = p2, so

     S12 = k1 * sqrt(p2) * S0 <= k1 * sqrt(p1) * S0 = S1.

   Hence the second gate is weakly redundant in the nested case.
